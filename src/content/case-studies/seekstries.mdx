Muscle-fibre images are a technical place to start learning image analysis. During my studies, I worked with a university team on a tool for CYTOO that could analyse many muscle-fibre images and identify their striations.

It was my first real application of image analysis and traditional machine learning. We had one full-time month to work in a domain that was new to all of us, so the task was as much about learning how to approach the problem as it was about building the application.

<ProjectImage src={striationImage} alt="Microscope image showing muscular fiber striations" caption="A microscopy image supplied for the project, where the fibre striations are the pattern the tool needed to identify." />

We built the interface in Python and used LBP patterns and k-means to recognise striations. The team then improved the approach with machine learning.

The project opened my mind to the range of problems image analysis can help solve. It also gave me a practical foundation for thinking about models as part of a usable application, not just an isolated technical exercise.
